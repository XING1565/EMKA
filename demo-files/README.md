# EMKA 面试演示文件

这个目录用于面试现场上传演示，覆盖文本、表格和图片三种多模态能力。

## 文件说明

| 文件 | 类型 | 展示点 |
| --- | --- | --- |
| `notes.txt` | 文本 | 项目背景、痛点、架构、边界，适合展示文本解析、分块和摘要 |
| `metrics.csv` | 表格 | 指标数据，适合展示表格解析、行范围引用和报告生成 |
| `diagram.png` | 图片 | EMKA 架构图，适合展示图片上传、图片元数据和 mock OCR 字段 |

## 推荐上传顺序

1. 上传 `notes.txt`
2. 上传 `metrics.csv`
3. 上传 `diagram.png`

## 推荐提问

```text
请总结我上传的资料，并引用来源。
```

```text
请根据指标数据生成一份 Markdown 报告。
```

```text
这些资料体现了 EMKA 的哪些技术亮点？
```
